drawCache

drawCache: boolean

是否缓存绘制命令（WebGL下有效），默认为false

description

stage设置绘制缓存后，当stage及其子元素相对与上一帧没有任何变化时，将直接向WebGL提交缓存的绘制命令，从而节省所有的中间计算过程。 对不经常变化的stage设置缓存后可以有效提高渲染效率，对经常变化的stage设置缓存会造成不必要的内存浪费
